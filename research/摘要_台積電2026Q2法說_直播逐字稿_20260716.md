## 一句話

**需求端：先進製程＋AI／HPC／Agentic AI 極強，全年美元營收成長「略高於 40%」，CapEx 升至 600–640 億美元。**  
**利潤端：Q3／下半年毛利被 2nm 爬坡稀釋約 3–4ppt，海外廠再稀釋 2–4ppt——這就是 ADR 跌的「刺」，不是訂單沒了。**  
**消費／價格敏感終端疲弱；成熟製程只有 AI 相關（PMIC、感測）緊，其餘不緊。**

## 1. 財務與指引（CFO｜Fact）

### 1.1 製程營收占比（Q2 晶圓營收）

| 製程 | 占比 |
|------|------|
| 2nm | **3%** |
| 3nm | **30%** |
| 5nm | **33%** |
| 7nm | **11%** |
| 7nm 及以下合計 | **77%** |

### 1.2 平台營收占比（Q2）

| 平台 | 占比 | QoQ |
|------|------|-----|
| **HPC** | **66%** | **+20%** |
| Smartphone | 22% | **-4%** |
| IoT | 5% | +4% |
| Automotive | 4% | +15% |
| DCE | 1% | +5% |

→ **HPC 已是絕對主引擎**；手機季減。

### 1.3 資產負債與現金（摘要）

- 現金及約當＋有價證券約 **3.5 兆台幣／1100 億美元**（口述 110 billion）  
- 庫存天數 **87 天**（+7），主因 **N2 爬坡**  
- 應收天數 29（+3）  
- Q2 營運現金流約 **7830 億台幣**；CapEx 約 **4960 億台幣／157 億美元**  
- 現金餘額季增約 990 億台幣至約 **3.1 兆**（與 3.5 兆口徑含有價證券需區分）

### 1.4 Q3 2026 指引（與 IR 表一致）

| 項目 | 指引 |
|------|------|
| 營收 USD | **44.6–45.8B**（中位約 QoQ +12%／YoY +37%） |
| 匯率假設 | 1 USD = **32** TWD |
| 毛利率 | **65–67%** |
| 營業利益率 | **56–58%** |

### 1.5 毛利為什麼 Q3 往下講（定價關鍵｜一手原話結構）

- Q2 毛利 **67.7%**，季增 **150bp**，略優於展望：成本改善＋利用率略升，**部分被海外廠稀釋抵銷**。  
- Q3 毛利中位約 **66%**（-1.7ppt）：主因 **2nm 陡峭放量稀釋約 3–4 個百分點**。  
- 下半年：2nm 稀釋仍約 **3–4ppt**。  
- 海外擴張規模變大：未來數年海外廠爬坡稀釋 **早期 2–3%、後期擴大到 3–4%**。  
- 對沖：領先製程需求很強＋持續成本／產能跨節點優化／生產力。  
- 匯率：無法控制的變數。

**對 ADR -4% 的一手印證**：公司自己把「毛利會被 2nm＋海外吃掉幾點」講死了；市場交易的是這個，不是營收指引。

### 1.6 CapEx 與股利

- **2026 全年 CapEx：600–640 億美元**（上修）  
- 結構：先進製程 **70–80%**；特殊 **~10%**；先進封裝／測試／光罩等 **10–20%**  
- 理由：結構性需求＋**Agentic AI**；與設備商提前備貨，**不預期擴產卡工具瓶頸**  
- 股利：2025 共 **18 元／股**；**2026 共 24 元／股**（+33%）；2027 預期持續提高  

### 1.7 CapEx 上修原因（Q&A 魏）

相對 1 月 52–56 → 4 月靠 56 → 現在 60–64：  
1. **客戶需求持續增加**，要台積一起把產能補上  
2. **通膨**：買設備的價格變貴

## 2. 魏哲家關鍵訊息（Fact／公司觀點）

### 2.1 需求雙軌

| 強 | 弱／審慎 |
|----|----------|
| AI 相關 **extremely robust** | 消費、價格敏感終端：零組件漲價＋總經不確定 → **challenged** |
| CSP 客戶 outlook 很正面 | 公司業務規劃 **prudent（審慎）** |
| 多年度 AI mega trend 信念 **very high** | |
| 全年美元營收成長 **slightly above 40%** | |

### 2.2 Agentic AI

- 代理式 AI → 資料中心 **CPU 角色回潮**，除加速器外再吃矽  
- 對台積正面：x86／ARM／RISC-V **幾乎都是台積客戶**  

### 2.3 產能與美國／台灣

- 領先製程設計導入＋放量 **>5 年**，無捷徑  
- 自上而下＋自下而上產能規劃  
- **亞利桑那追加 1000 億美元**：更多 **2nm 及以下** 邏輯廠＋**先進封裝**，服務美系大客戶  
- 同時台灣未來數年建 **13 座**領先製程與先進封裝廠，**持續投資台灣**  
- N3：台／美／日 **各再加一座 3nm 廠**；台灣 N5 設備轉 N3；跨 N7/N5/N3 彈性  

### 2.4 成熟製程（Q&A 講死）

- **只有跟 AI 相關的成熟製程緊**：首要 **電源管理 IC（PMIC）**（0.18µm、~19nm 等）；再來 **感測器**  
- **其他消費向成熟製程需求不強**，並非全面缺貨漲價  

### 2.5 A14

- 第二代 nanosheet；相對 N2：同功耗 **+10–15% 速度** 或同速度 **-25–30% 功耗**，密度約 **+20%**  
- 開發進度佳；客戶 smartphone + HPC AI 興趣強；tape-out 超前  
- **2027 風險試產、2028 量產**  
- 家族延伸 A13／A12（口述後段）  

### 2.6 競爭（天下曾摘「買牛奶」；直播結構）

- 選代工伙伴 **≠ 超商換店**；技術合作、試產、備產能到放量約 **五年**  
- 技術、製造、客戶信任；對記憶體廠高毛利「嫉妒」屬情緒，不改上述本質

## 3. 我們的判斷（更新）

| 維度 | 結論 |
|------|------|
| 訂單能見度 | **上調**（HPC 66%、全年 >40%、CapEx 被客戶推著加） |
| 毛利／FCF 敘事 | **短線有刺**（2nm 3–4ppt + 海外 2–4ppt）→ 解釋 ADR |
| 消費電子 | 公司自認疲弱；別指望手機季救估值 |
| 成熟製程題材 | **僅 PMIC／感測等 AI 外溢**；禁止「成熟全面缺貨」幻想 |
| 操作 | 維持 **HOLD／回檔分批**；一手證實期權前瞻「CapEx 壓利潤率」敘事 |

**Fact 升級**：CapEx 60–64B、全年 slightly above 40%、2nm 稀釋 3–4ppt、海外 2–4ppt、HPC 66%、PMIC 缺——可自本直播當一手引用。
