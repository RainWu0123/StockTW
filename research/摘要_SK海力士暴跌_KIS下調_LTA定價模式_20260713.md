## 二、事件背景：為什麼跌

### 五重因素疊加

| 因素 | 說明 |
|------|------|
| **1. KIS 下調盈利預期** | Q2 營業利潤預估 60.4 兆韓元，低於市場共識 65 兆韓元（差距約 8%）。同步下調 2026/2027 年盈利預期 9% 和 11% |
| **2. ADR 上市利好兌現** | 7/10 海力士 ADR 以 149 美元發行，首日收 168.01（+12.8%），募资 265 億美元。上市完成 = 事件驅動資金獲利了結 |
| **3. 韓美溢價套利** | ADR 首日相對韓股存在約 15% 溢價 → 套利資金賣韓股買 ADR → 韓股承受額外賣壓 |
| **4. 槓桿產品機械式殺盤** | 兩倍做多海力士 ETF 跌近 20%，每日再平衡 → 機械式追殺放大跌勢 |
| **5. 板塊性去槓桿** | 三星同跌 6%+ 說明不是海力士單獨爆雷，而是韓國 AI 資產集中去槓桿 |

## 三、KIS 研報的核心洞察：LTA 定價模式變遷

**這是本文最有價值的部分。**

### 表面：「盈利不及預期」
### 實質：存儲行業商業模式正在改變

KIS 下調的不是基本面，是估值模型。核心邏輯：

| 傳統存儲週期 | AI 時代存儲模式 |
|-------------|----------------|
| 高度依賴現貨價格 | 長期供貨協議（LTA）鎖定 |
| 價格暴漲 → 利潤暴增 | LTA 提前鎖定價格 → 利潤峰值被平滑 |
| 週期性强，暴起暴落 | 需求確定性提升，盈利持續性增 |
| 盈利彈性高但不可持續 | 盈利峰值降低但持續更久 |

**具體機制**：
- 海力士 HBM 訂單大部分已透過 LTA 提前鎖定價格
- 傳統 DRAM/NAND 現貨價 Q2 大幅上漲（DRAM +30%，NAND +50% 環比）
- 但海力士 HBM 的 ASP 因 LTA 鎖價，無法完全享受現貨暴漲的邊際收益
- 三星因 HBM 長協比例較低，反而更能享受現貨暴漲 → 市場開始下調海力士相對三星的短期盈利彈性

**關鍵數字**：
- KIS 下調後仍預計海力士 2026 Q2 營業利潤率 **74.6%**（歷史極高）
- KIS 維持目標價 **380 萬韓元**（非常價 184.5 萬韓元仍有 ~106% 上行空間）
- 2026/2027 年盈利預期分別下調 9% 和 11%，但核心原因不是需求放緩而是估值模型調整

### 對台股研究的連結

> LTA 定價模式改變了存儲股的估值邏輯。過去看存儲看現貨價格，未來要看 LTA 條款 + HBM 放量節奏 + Big Tech CAPEX 持續性。

## 四、存儲週期見頂了嗎？

### 頂部確認的六個訊號（本文列出）

| 訊號 | 目前狀態 |
|------|---------|
| 1. DRAM/NAND 合約價由漲轉跌 | ❌ 未出現。Q3 合約價仍漲（DRAM +13-18%，NAND +10-15%） |
| 2. 原廠/通路庫存持續增加 | ❌ 未出現 |
| 3. 客戶取消或推遲訂單 | ❌ 未出現 |
| 4. 大型雲端商下調 AI CAPEX | ❌ 未出現（Meta CAPEX 1250-1450 億美元仍增） |
| 5. 新增產能釋放超過需求 | ⚠️ 需關注 Q4 擴產節奏 |
| 6. HBM 與傳統 DRAM 利潤率同步下滑 | ❌ 未出現 |

**三個層次必須區分**：
1. 存儲**價格漲幅**接近頂部 → 可能
2. 企業**盈利增速**接近頂部 → 可能
3. 存儲**上行週期**結束 → 尚未
