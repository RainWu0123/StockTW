## 2. 我們的方式（ERA + TSA 紀律）

### 2.1 Fact / Consensus / Estimate 拆層

| 層 | 內容 |
|----|------|
| Fact | Q2 營收／利潤／EPS／毛利率／mix；Q3 官方區間；2nm 開始貢獻 3% |
| Consensus | CapEx 600–640B、全年 >40%、美再投千億、A14 時程（天下／法說口述） |
| Estimate | 「EPS 破百」、合理本益比、短線是否利多出盡 |

### 2.2 營收／訂單能見度

- **能見度：高**。Q2 達標上緣＋Q3 指引再跳一截＋全年成長上修，不是「財報好、指引爛」的博通式劇本。  
- 驅動：領先製程＋AI/HPC；2nm 進入陡升。  
- 稀釋：毛利指引 Q3 回落至 65–67%＝**承認 2nm 爬坡＋海外成本**，屬健康誠實，不是需求轉弱。

### 2.3 與「利多鈍化」框架對照（三星案例）

| 條件 | 三星 Q2 記憶體 | 台積 2026Q2 |
|------|----------------|-------------|
| 財報 | 暴增 | 暴增 |
| 指引 | 市場解讀轉弱／鈍化 | **明確上修營收與 CapEx** |
| 股價當下 | 大跌／熔斷鏈 | ADR 盤前弱；台股現貨日收 **+1.23% 至 2470**（live） |

→ 同屬「高估值環境」，但台積是 **Beat + Raise**，不是 Beat 後指引掉鏈。  
短線仍可能「買預期、賣事實」波動；**中線基本面論證強化**，與三星頂部確認不同級。

### 2.4 估值（roughly right，禁單點算命）

以 7/16 現價 **2,470**、若全年 EPS 粗到 **~100**（媒體推導，Estimate）：

| 情境 | 假設 | 隱含 |
|------|------|------|
| 若 EPS 100、PE 25x | 保守倍數 | 股價 ~2,500 |
| 若 EPS 100、PE 30x | 接近現水平 | ~3,000 |
| 若 EPS 90、PE 28x | 略保守 EPS | ~2,520 |
| 若 EPS 110、PE 32x | 樂觀 | ~3,520 |

**解讀**：若市場接受「EPS 破百」敘事，現價並未給極端泡沫倍數；但 **Forward 已不便宜**，上修若已部分反映，ADR 弱＝邊際買盤要求「再驚豔一點」。  
（舊檔 7/14 用 PE 32.8×2420 隱含 EPS ~74 為近用／TTM 口徑，與「全年 100」不可混用同一句——**TTM vs 當年**要分開講。）

### 2.5 劇本更新（1D-1W-1M）

| 欄位 | 更新 |
|------|------|
| 持有價 | 長線核心：法說後**論證加強**，可持 |
| 分批買 | 原 2250–2300／2100–2200 仍有效；**強指引後可把第一筆上移觀察至 2350–2450 回檔區**，但不追缺口 |
| 合理帶 | Base 上修敘事：約 **2,400–2,900**（視 EPS 共識收斂） |
| 防守 | 短線：放量跌破 **2,300** 減波段；系統性：費半破位＋ADR 連殺 |
| 賣出 | ① 下季指引大幅下修 AI 語氣 ② 關鍵客戶公開砍單 ③ CapEx 暴砍且訂單能見度同步壞 |

**TSA 判決（操作）**：**HOLD**（已有倉可持／回檔加；無倉不追高）。  
非 YES 追價、非 NO 看空基本面。

### 2.6 風險（可觀測）

1. Q3 毛利落在指引下緣＋市場解讀「海外稀釋失控」  
2. CapEx 600–640B 推升折舊，自由現金流敘事惡化（對齊 AI 債／CSP 現金流討論）  
3. 地緣／美國投資政治化成本  
4. 記憶體鏈續殺 → 半導體 beta 拖累，與基本面脫鉤下跌  
5. 2nm 良率／爬坡不如預期

### 2.7 Sell-side 讀法（本庫檢查表）

- **不要**因「某家目標價上修到 XXX」就改劇本。  
- 應盯：全年成長共識是否從 ~30% 級抬到 **>40%**；CapEx 共識是否同步；毛利稀釋是否被消化。  
- 目標價權重最低；**Facts＝本檔第 1 節**。

### 股癌式白話（角色濃縮，非逐字孟恭）

- 這次不是「看圖說故事」，是**真金白銀季報＋把全年與 CapEx 往上抬**。  
- 但市場若已買很滿，**好消息可以跌**——灰階，不是被背叛。  
- 記憶體那邊殺估值，不自動等於台積需求沒了；要分開供需層。  
- 菜雞別因法說 ALL-IN 融資；腳尖。真長線，回檔再談。  
- 外面中小題材炸完，資金回流神山的劇本仍在，但那是**條件句**，不是每天的操作指令。

### 產業輔對照（本地 2026 節目線）

| 過去節目敘事 | 本次法說 |
|--------------|----------|
| CapEx 上修驅動設備鏈（EP628 等） | 再上修到 600–640B 敘事 → 設備／廠務邏輯 theoretically 延續 |
| 萬箭歸宗／外資賣壓混沌盤 | 法說強化「歸宗基本面」；短線外資／ADR 仍可壓 |
| 成熟製程／被動／CPU 外溢 | PMIC 缺貨＝AI 外溢到成熟，題材有原型，仍要營收占比 |
| 見好不收 vs 打臉 | 指引上修＝暫時不必打臉多頭基本面；估值太兇仍要彈性 |

## 4. 我們 vs 股癌：哪裡不同、哪裡相同

| 維度 | 我們的方式（ERA/TSA） | 股癌方式（gooaye） | 差異本質 |
|------|----------------------|-------------------|----------|
| 輸出目標 | 可落盤數字、六欄位、HOLD/YES/NO | 心態＋判讀＋供需分層；少給精確點位 | 我們偏**可執行研究**；他偏**防呆與敘事衛生** |
| 資料紀律 | Fact/Consensus/Estimate；官方 PDF 優先 | 食物 vs 食品；一手優先（同方向） | **相同**，用語不同 |
| 估值 | Bear-Base-Bull、禁單點目標價 | 高倍數要可見數字；rough 心法 | 我們更**表格化**；他更**原則化** |
| 時間尺 | 強制 horizon 1D-1W-1M vs 長線 | 長短要用不同腦（同 Vincent／我們） | 相同 |
| 操作 | 分批價、防守、賣出條件 | 腳尖、對帳單、反新手槓桿、歸宗條件句 | 我們給**價區**；他給**行為約束** |
| 對「ADR 跌」 | 定價機制：預期差／利多鈍化／beta | 標題別蓋過財報；灰階接受好消息跌 | 相同現象，我們多**機制拆解** |
| 對記憶體 | 分開 TSMC 與記憶體週期；交叉頂部研究 | 供需分層；別齊殺齊買 | 相同 |
| CapEx 上修 | 能見度↑但折舊／FCF／債市約束 | 設備鏈題材輔；見好不收要搭配風險 | 我們多接 **AI 債／CSP 現金流**總經 |
| 禁止 | 目標價迷信、score 當下單 | 萬用買點、融資梭哈、偽推中小 | 相同精神 |
| 角色 | 研究員／決策卡 | 主委心法（可粗） | 語氣 |

### 合成結論（給製作人）

1. **基本面共識**：雙方都不會把這次法說讀成「需求崩」。  
2. **操作共識**：都不支持無腦追高；都強調回檔與風險預算。  
3. **我們多做的**：Q3 指引 vs 前次、mix、估值表、劇本數字、與三星／記憶體／晨報交叉。  
4. **股癌多做的**：強迫你看自己對帳單與槓桿；防止標題「ADR 跌」洗掉判讀；歸宗是條件不是信仰。  
5. **合併使用**：用我們的表決定**加不加、哪裡加**；用股癌檢查**該不該動、動的時候手有沒有在發抖**。
