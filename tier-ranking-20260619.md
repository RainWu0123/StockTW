## 1. 核心選股邏輯
### 從「營收確認型」到「動能+題材先行型」的修正

**事故檢討**：華邦電過去寫成 90 元、南亞科 435 元、國巨 1000 元，皆為舊資料。
- 根本原因：未在执行研究报告前先跑 TWSE API 即時快照，用记忆中的舊估值建立偏误
- 新規則：寫研究报告**第一步永遠是 TWSE API 快照**

### 層次架構

#### 第一層：題材純度篩選（只選有營收落地時間表的）
- AI GPU / ASIC → 已經發生（國巨、台積電、聯發科）
- AI DRAM / HBM → 已經發生（南亞科、華邦電）
- AI 散熱 / 液冷 → 已經發生（奇鋐、雙鴻）
- CoWoS / ABF 載板 → 已經發生（日月光、欣興、南電）
- CPO / 矽光子 → 已經發生（光聖、聯電）
- AI Edge / 機器人 → 2026 下半年（凌華）
- SiC / GaN → 2026 下半年 — 2027（環球晶）
- 地緣政治（鎢 / APT）→ 即時反彈（聯友金屬）

#### 第二層：動能確認（TWSE 即時數據）
```
營收年增 > 50% + 法人連續買超 > 3 天 + 股價處於 52 周高區 20% 內 = 強動能
營收年增 20-50% + 法人買超 + 股價創新高 = 動能
營收年增 < 20% + 法人買超 = 觀察，不追高
```

#### 第三層：賣出時機
```
1. 單日成交是 5 日均量 2 倍以上 + 上影線長度 > 實體 K 線 → 隔日开盘先賣
2. 外資買超連續 3 天 > 5 萬張，第 4 天縮減至 1 萬張以下 → 法人獲利了結
3. 月營收低於預期 15% → 不戀棧
4. 地緣題材股（聯友金屬）→ 消息面鈍化即刻出
```

#### 第四層：配置原則
```
50%：營收確認型（國巨、聯發科、台積電、台光電、台達電）持有 7-30 天
30%：動能爆發型（DRAM 雙雄、凌華、聯電、奇鋐、日月光）持有 1-7 天
20%：地緣題材型（聯友金屬）持有 1-3 天
```
